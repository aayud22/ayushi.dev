# Ayushi's Developer Portfolio 🚀

A sleek, high-performance personal portfolio built to showcase my web development projects and frontend engineering skills. Designed with a minimalist, high-contrast aesthetic, featuring smooth scroll animations, a responsive grid layout, and a fully functional serverless contact form.

## ✨ Features

- **Modern Architecture:** Built on Next.js (App Router) and React for blazing-fast performance and SEO optimization.
- **Premium UI/UX:** Styled entirely with Tailwind CSS, featuring a custom animated cursor and sleek custom scrollbars.
- **Fluid Animations:** Integrated Framer Motion for buttery-smooth scroll reveal animations and interactive hover states.
- **Secure Contact API:** Custom backend API route utilizing Nodemailer, fully secured with React Hook Form and Zod payload validation.
- **Resilient Design:** Built-in fallback UI components for robust error handling on external assets.

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Forms & Validation:** [React Hook Form](https://react-hook-form.com/) + [Zod](https://zod.dev/)
- **Email Handling:** [Nodemailer](https://nodemailer.com/)

## 🚀 Running Locally

1. Clone the repository:
   \`\`\`bash
   git clone https://github.com/aayud22/ayushi.dev.git
   \`\`\`
2. Install dependencies:
   \`\`\`bash
   npm install
   \`\`\`
3. Set up your `.env.local` file with your SMTP credentials for the contact form.
4. Start the development server:
   \`\`\`bash
   npm run dev
   \`\`\`
